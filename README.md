# Offres_service

Microservice des offres

Lancer le microservice discovery-server avant de lancer ce microservice.
Pour lancer ce microservice, il faut :

# Console :

    cd Offres/
    Run gradle bootRun

# Intellij :

    Build le projet
    Onglet Gradle > Offres > Tasks > application > bootRun
